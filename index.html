Notes from Course 0

History of HTML (hypertext markup language): invented in 1990s

You (browser) <=> Internet <=> Servers (via HTTP)

HTML consists of:
  • Text content (what you see)
  • Markup (what it looks like)
  • References to other documents
  • Links to other pages

    • Tags (opening and closing)
    Opening Tag: <NAME>
    Closing Tag: </NAME>

    Elements: String of text includings tags and content

    Other tags:
    <b> = bold
    <em> = italics

    HTML Attributes
    <TAG ATTR = "value"> content </TAG>

    Anchors: links
    <a href ="link"> content </a>

    <img src = "url" alt = "text"> (VOID tag)

    Void Tags: tags that do not require closing tag

    Whitespace
    <br> = break (VOID tag)
    <p> = paragraph

    Inline vs. Block tags
    Inline - work within existing text (bold, italics, line breaks, etc.)
    Block - create imaginary box that contains everything until closing tag (paragraph, form, div, etc.)

    HTML Document Example

    <! DOCTYPE HTML>
    <html>
    <body>
    <head>
    <title> TITLE! </title>
    </head>
    <body>
    <b>Contents</b>
    </body>
    </html>
